# nico-slide
niconico風にツイートを取得して来て表示するデスクトップアプリ

とりあえず動く物が欲しかっただけなのでガバガバです。

内容としてはwebviewで内部でtweetdeckを開き、tweetdeckの1列目のツイートを毎秒拾いに行き、デスクトップ上に表示する。
という物です。
twitter自体に負荷掛けている訳でも、APIの不正利用でもないです。
